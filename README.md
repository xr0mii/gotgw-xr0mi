# xr0mi-adv: Вина Могильного Мира

Русская локализация материалов приключения **Starfinder Second Edition: Guilt of the Grave World** для Foundry VTT.

Модуль предназначен для использования вместе с официальным модулем **sf2e-guilt-of-the-grave-world**. Для корректной работы требуется установленная официальная версия приключения, а также система **Starfinder Second Edition**.

Текущая сборка содержит перевод журнальных материалов. Актёры, предметы и сцены могут быть добавлены или доработаны отдельно.
