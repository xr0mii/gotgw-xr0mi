# Вина Могильного Мира: русский перевод

Личный модуль перевода для `sf2e-guilt-of-the-grave-world`.

Модуль не должен содержать оригинальные изображения, карты, аудио и другие ассеты Paizo. Документы могут ссылаться на пути официального модуля, например `modules/sf2e-guilt-of-the-grave-world/assets/...`, поэтому для использования нужен купленный и установленный официальный модуль.

## Структура

- `packs/journals` - переведенные JournalEntry.
- `packs/scenes` - переведенные Scene.
- `packs/actors` - переведенные Actor.
- `packs/items` - переведенные Item.
- `packs/adventure` - финальный Adventure pack, когда все будет собрано в Foundry.
- `exports` - рабочие JSON-экспорты до импорта обратно в Foundry.
- `translations/glossary.json` - локальный глоссарий приключения.
- `tools` - будущие скрипты проверки и переноса перевода.

См. также `docs/foundry-pack-structure.md`: официальный модуль хранит итоговое приключение как один Foundry LevelDB pack типа `Adventure`. Раздельные pack-и в этом модуле нужны как рабочая зона для импорта/экспорта, а финальную сборку приключения должен делать Foundry.

## Правила перевода JSON

Не менять технические поля Foundry и SF2E: `_id`, `uuid`, `flags`, `system`, `sort`, `ownership`, `permission`, `folder`, `type`, `img`, `texture`, пути к ассетам и любые числовые настройки.

В HTML и описаниях сохранять:

- Foundry-ссылки: `@UUID[...]`, `@Compendium[...]`, `@Check[...]`, `@Damage[...]`, `@Template[...]`, `@Localize[...]`.
- Inline rolls: `[[...]]`.
- CSS-классы и структуру тегов.
- Названия compendium pack и id документов внутри ссылок.

Переводить только пользовательский текст: `name`, заголовки, содержимое journal pages, описания, подсказки, boxed text и видимый текст внутри фигурных скобок Foundry-ссылок, если сама ссылка остается прежней.

Основной источник терминов - `pf2e-ru`, особенно `pf2e-ru/data/community/sf2e`.
