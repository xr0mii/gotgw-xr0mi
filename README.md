# xr0mi-adv: Вина Могильного Мира

Русская локализация материалов приключения **Guilt of the Grave World** для Foundry VTT.

Модуль предназначен для использования вместе с официальным модулем **sf2e-guilt-of-the-grave-world**.

Текущая сборка содержит перевод журнальных материалов. Актёры, предметы и сцены могут быть добавлены или доработаны отдельно.

Связаться со мной: [boosty.to/xr0mi](https://boosty.to/xr0mi)
